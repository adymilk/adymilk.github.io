Swiper.js

强大的移动端触摸欢动插件Swiper


Swiper常用于移动端网站的内容触摸滑动.Swiper是纯javascript打造的滑动特效插件，面向手机、平板电脑等移动终端。Swiper能实现触屏焦点图、触屏Tab切换、触屏多图切换等常用效果。Swiper开源、免费、稳定、使用简单、功能强大，是架构移动终端网站的重要选择！
